# 2026-03-16

## OpenClaw Pay — AI 原生加密货币支付协议

### 项目创建
- 从零创建了完整的 OpenClaw Pay 项目
- 技术栈: TypeScript + ethers.js + WebSocket + Express

### 核心架构
- **协议层** (`src/protocol/`): PaymentIntent, AuthorizationRequest, Policy 类型定义 + IntentManager + AuthManager
- **钱包层** (`src/wallet/`): KeyVault (AES-256-GCM 加密) + TxSigner (交易构建签名)
- **Agent SDK** (`src/agent/`): OpenClawAgent — AI Agent 只需调用 `agent.pay()` 即可发起支付
- **授权服务器** (`src/server/`): WebSocket + REST API 中间层
- **钱包 App** (`wallet-app/`): Web 界面，用户审批支付请求

### 设计原则
- AI Agent 永远不接触私钥
- 支持三种授权模式: 单笔授权、预授权(额度限制)、白名单
- WebSocket 实时推送授权请求
- 交易由钱包端签名后上链
